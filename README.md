1. Description
This is a simple online chat application built using Java socket programming.
It allows multiple users to connect to a central server and exchange messages.

2. Requirements
- Java JDK 8 or higher
- Command Prompt / Terminal

3. How to Run

Step 1: Compile the server
javac ChatServer.java

Step 2: Run the server
java ChatServer

Step 3: Compile the client
javac ChatClient.java

Step 4: Run multiple clients (open multiple terminals)
java ChatClient

4. Features
- Multi-client support
- Unique user IDs
- Message broadcasting
- Text-based user interface
